Civic Chat Widget

Basic Integration Example

Widget sollte erscheinen:

Unten rechts sollte eine Chat-Box erscheinen - sofort sichtbar ohne Klick!

Sofort startbereit

Chat-Box erscheint automatisch - kein Button-Klick nötig

Voll funktionsfähig

Minimieren, schließen, Nachrichten senden

Responsive

Passt sich an alle Bildschirmgrößen an

Einfach integrierbar

Nur wenige Zeilen Code nötig

Integration Code

<!-- Civic Chat Widget --> <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/johanna1105/civic-chat-widget@main/dist/widget.min.css"> <script src="https://cdn.jsdelivr.net/gh/johanna1105/civic-chat-widget@main/dist/widget.min.js"></script> <div id="civic-chat"></div> <script> window.CivicChat && window.CivicChat.init(); </script>

Test-Anleitung:

  1. Chat-Box sollte unten rechts erscheinen
  2. Klicke auf "−" zum Minimieren
  3. Klicke auf "×" zum Schließen
  4. Zum Verschieben: Header greifen und ziehen
  5. Teste das Senden einer Nachricht
  6. Öffne Browser Console (F12) für Debug-Info

Erweiterte Konfiguration

<div id="civic-chat" data-title="Mein Support" data-welcome-message="Hallo! Wie können wir helfen?"> </div>
